Detailed Description

A class implementing an Weld joint.

The underlying implementation in Simbody is a MobilizedBody::Weld.

void WeldJoint::createSystem ( SimTK::MultibodySystem &  system) const [protected, virtual]

createSystem() is called when the SimTK::System is being created for the Model.

It must be implemented in order to add appropriate SimTK elements to the System corresponding to this component. Helper methods for adding modeling option, state variables and their derivatives, discrete variables, etc., are available and can be called within createSystem() only.
